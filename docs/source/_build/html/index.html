Welcome to Aquarius ONE SDK documentation!

Aquarius ONE SDK is a C# SDK for .NET enabling developers to interact with the Aquarius ONE Platform. Documentation for the RESTful APIs can be found at: Aquarius ONE Developer Portal <https://aqi-feature-api-mgmt.developer.azure-api.net/>.

Check out the Usage section for further information, including how to Installation the nuget package.

Note

This project is under active development.
