Enterprise Data APIsÔÉÅ

Provides data proxy to abstract the interaction with cloud enabled data sources through data adapters. This service is leveraged by Enterprise Report to be able to assemble information across a variety of data sources. Enterprise Twin is leveraged as a data catalogue that also provides resource-based access control through Telemetry Twins.
