Qualifiers

This is a static helper class to assist with laboratory qualifiers.

Class

class ONE.Operations.Spreadsheet.Qualifiers

Concepts

Cascade rule

CASCADE RULE: Sets when to display the data qualifiers (<,>, ND) with the calculated statistic. Valid settings are:

  1. Display no data qualifiers in result.

  2. Display the data qualifier if one of the values contains the qualifier.

  3. Display data qualifier if over half the values contains the qualifier.

  4. Display data qualifier if all of the values contains the qualifier.

Data Qualifier rules

  1. Zero

  2. Entered Value

  3. Half of <, twice of >
